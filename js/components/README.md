# JS Components

Ce dossier contiendra les composants d'interface de MeteoSignal.

Modules prévus :

- `header.js` ;
- `search.js` ;
- `current-weather.js` ;
- `weather-cards.js` ;
- `hourly-forecast.js` ;
- `daily-forecast.js` ;
- `astronomy.js` ;
- `favorites.js`.

Chaque composant reçoit des données préparées et met à jour uniquement sa zone de l'interface.
