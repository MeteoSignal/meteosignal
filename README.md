# 🌦️ MeteoSignal

> **La météo, partout avec vous.**

MeteoSignal est une **Progressive Web App (PWA)** moderne, rapide et évolutive, conçue pour afficher des informations météorologiques fiables sur **smartphone, tablette Android, ordinateur et télévision**.

Le projet est développé en **HTML, CSS et JavaScript** avec une architecture pensée pour évoluer facilement et intégrer plusieurs fournisseurs de données météo.

---

## 🎯 Mission

Offrir une application météo **simple, rapide, moderne et accessible**, quel que soit l'appareil utilisé.

---

## ✨ Fonctionnalités actuelles

* ✅ Météo actuelle
* ✅ Prévisions sur 7 jours
* ✅ Horloge temps réel
* ✅ Progressive Web App (PWA)
* ✅ Compatible GitHub Pages
* ✅ Interface responsive (en évolution)
