# 🌦️ MeteoSignal

> **Modern Weather Progressive Web App**

MeteoSignal est une **Progressive Web App (PWA)** moderne dédiée à la météo.
Le projet est conçu pour fonctionner sur **smartphone, tablette Android, PC et TV**, avec une interface responsive, rapide et agréable à utiliser.

Le but est de développer une application météo complète, évolutive et open source, en utilisant principalement **HTML, CSS et JavaScript**.

---

# ✨ Fonctionnalités

## ✅ Disponibles

* Météo actuelle
* Prévisions sur 7 jours
* Horloge temps réel
* Installation en PWA
* Compatible GitHub Pages
* Interface responsive
* Version de l'application affichée

---

# 🚀 Installation

Clonez le dépôt :

```bash
git clone https://github.com/MeteoSignal/meteosignal.git
```

Puis ouvrez simplement `index.html` dans votre navigateur.

Aucune installation supplémentaire n'est nécessaire.
